2. Description du Service

  1. Scanium est une application Android qui :
    1. Utilise l'apprentissage automatique sur l'appareil pour détecter des objets physiques via l'appareil photo de l'appareil.
    2. Fournit des fourchettes de prix estimées basées sur les catégories d'objets.
    3. Prend en charge la détection d'objets, le scan de codes-barres/QR et la reconnaissance de texte de documents.
    4. Permet aux utilisateurs de gérer les articles détectés localement sur leur appareil.
    5. Comprend un flux de vente simulé sur un marché (par exemple, flux de type eBay), sans effectuer de transactions réelles.
  2. L'Application fonctionne hors ligne par défaut et est axée sur la confidentialité :
    1. Par défaut, tout le traitement se fait localement sur l'appareil de l'utilisateur.
    2. La classification basée sur le cloud est facultative et n'est activée que si elle est explicitement configurée par l'utilisateur ou le développeur via des variables d'environnement ou une configuration locale.
